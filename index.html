Crazy Ostrich


  • High concept
    Crazy Ostrich is a third-person 3D platformer in which you control a fast ostrich accross a puzzly corral. The player must must avoid hitting wooden objects and complete different types of objectives before time runs out.
  • Release year
    2014
  • Project length
    6 months
  • What I did
    - Game design;
    - Game art;
    - Game programming;
    - SFX.
  • Engine
    Blender Game Engine
  • Coded in
    Blender Logic Bricks
  • Relevant links
    - Download page: https://richarddiasribeiro.itch.io/crazy-ostrich;
    - Repository: https://github.com/richarddiasribeiro/Crazy-Ostrich.
  • Personal notes
    This was my very first fully developed game. My main challenges with this one were probably dealing with collision (at least back in the day (2014) collision in Blender was a little tough to get right, as objects often went through one another unwantedly). Also, through this project, I've learned the importance of avoiding "fake difficulty", especially related to player mechanics (it was clear right from the start that poor controls could easily make the game feel frustrating).
